# Web Note Editor

웹 브라우저에서 사용할 수 있는 메모장 서비스 소스코드를 담은 모놀리스 리포지토리입니다.
Express 기반의 백엔드와 React 기반의 프론트엔드 애플리케이션을 포함합니다.
두 애플리케이션은 각각 `/backend` 디렉터리, `/frontend` 디렉터리에 위치합니다.

- React 프론트엔드 앱: ([/frontend#readme](./frontend#readme))
